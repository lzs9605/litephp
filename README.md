LitePhp的公共库，需先创建LiteApp 或LiteApi项目使用



## 主要新特性

* 采用`PHP7`强类型（严格模式）
* 支持更多的`PSR`规范
* 原生多应用支持
* 对IDE更加友好
* 统一和精简大量用法


> LitePhp 1.0的运行环境要求PHP7.0+，兼容PHP8.1

## 安装

~~~
composer require unntech/litephp
~~~


如果需要更新框架使用
~~~
composer update unntech/litephp
~~~
